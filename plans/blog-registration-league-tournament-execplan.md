# Online Registration for League or Tournament Guide

## Goal

Create the BracketIQ guide `How to Set Up Online Registration for a League or Tournament` as the next product-led guide from the roadmap.

## Scope

- Publish a guide under `/guides/registration-league-tournament`.
- Classify the guide under Events because it is a cross-format registration workflow that supports leagues and tournaments.
- Keep the guide focused on the common registration process: registration mode, cutoffs, capacity, price, public page review, captain team selection, free-agent access, and organizer participant follow-up.
- Link to the deeper league and tournament registration guides for format-specific setup details.
- Use indoor soccer fixtures and real local BracketIQ screenshots.

## Workflow

1. Reuse the existing league and tournament registration fixtures.
2. Capture screenshots from the rendered app for each registration step.
3. Write end-user instructions against those screenshots.
4. Register the guide metadata, guide nav order, sitemap expectations, and roadmap status.
5. Run focused blog/guide tests, TypeScript, and a rendered route smoke test.
