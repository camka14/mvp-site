# League Playoffs Guide

## Goal

Create the BracketIQ guide `How to Run a League With Playoffs` as the next League guide after league creation, management, multi-week scheduling, and standings/playoff seeding.

## Scope

- Use an indoor soccer league as the generic sport example.
- Show the real BracketIQ workflow for confirming playoff setup, reviewing the qualified teams, checking Agenda view, confirming standings, verifying seeded playoff brackets, and checking participant-facing playoff information.
- Use Agenda view for schedule screenshots.
- Capture screenshots from the rendered local app.
- Register the guide under `/guides/league-playoffs`.
- Update the roadmap and focused tests.

## Workflow

1. Reuse or reset the local indoor soccer playoff league fixture so the guide can show a league with completed regular-season results and playoff bracket slots.
2. Capture the guide steps from the rendered app.
3. Write the end-user instructions against those screenshots.
4. Add blog registry metadata, guide nav order, sitemap expectations, and roadmap status.
5. Run focused blog/guide tests, typecheck, and a rendered route smoke test.
